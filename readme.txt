Esta página web es para el manejo de una plataforma educativa web.
Se pueden ingresar datos de estudiantes, materias y familiares de los estudiantes a través de formularios que se encuentran en el pie de página, a los que somos dirigidos haciendo click en los botones de la barra de navegacion.
Estudiante: tiene los nombres, carreras y edades de los estudiantes de la academia.
Materias: se muestran las materias ofrecidas, con el area educativa y los créditos asociados. Se cuenta con un buscador para buscar si existe una materia con el nombre deseado.
Familiares: tiene los datos de las familias de los estudiantes para el uso de profesores.
Además, al ingresar a "http://127.0.0.1:8000/admin/" se puede ingresar con el usuario.
